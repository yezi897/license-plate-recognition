# 前端重构设计文档

## 概述

将车牌识别系统前端从当前的基础样式重构为现代化、美观的界面，提升视觉体验和交互质感。

## 设计决策

### 视觉风格：简约白

- 白色背景为主体
- 柔和阴影（`0 1px 3px rgba(0,0,0,0.06)` 到 `0 4px 14px rgba(37,99,235,0.25)`）
- 大圆角卡片（12px border-radius）
- 系统字体栈：`-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`

### 布局方案：侧边导航

- 左侧 64px 宽图标导航栏
- 右侧主内容区
- 导航栏包含：Logo、实时监控、识别历史、设置
- 当前激活项用蓝色背景高亮

### 强调色：科技蓝

- 主色：`#2563eb`
- 渐变：`linear-gradient(135deg, #1d4ed8 0%, #3b82f6 50%, #60a5fa 100%)`
- 用于：按钮、车牌号高亮、导航激活态、链接

### 按钮设计

#### 主按钮（拍照识别）

- **默认态**：渐变蓝背景 + 呼吸脉冲光环动画 + SVG 图标
- **Hover**：上浮 2px + 阴影加深 + 倾斜 25° 光泽扫过动画
- **点击态**：微微下沉

#### 次级按钮（自动识别）

- **默认态**：白色背景 + 灰色边框 + SVG 图标
- **Hover**：上浮 2px + 阴影加深 + 边框变深

#### 状态变化

- **识别中**：旋转加载圈 + 底部进度条动画 + 其他按钮置灰禁用
- **自动模式启动**：按钮变为红色停止按钮（`#ef4444`）+ 红色脉冲 + 绿色呼吸指示灯
- **自动模式停止**：恢复默认次级按钮样式

#### 图标

使用 SVG stroke 风格图标（Feather Icons 风格）：
- 拍照识别：相机图标
- 自动识别：太阳/光芒图标
- 暂停：双竖线图标
- 停止：双竖线图标（红色）

### 识别结果卡片

- 车牌号：28px，科技蓝，等宽字体，字间距 2px
- 颜色/置信度/时间：13px，水平排列
- 置信度用绿色（`#16a34a`）显示

### 统计卡片

- 今日识别次数：22px 深色数字
- 平均置信度：22px 绿色数字
- 两张卡片并排

### 历史表格

- 表头灰色背景（`#f8fafc`）
- 车牌号列：科技蓝 + 等宽字体 + 600 字重
- 置信度列：绿色
- 行分隔线：`#f1f5f9`
- 底部分页居中

### 设置弹窗

- 居中弹窗，白色背景，12px 圆角
- 半透明黑色遮罩
- 表单输入框：聚焦时蓝色边框 + 蓝色阴影

## 技术方案

### 文件结构

保持现有三文件结构，不引入框架：
- `web/index.html` — HTML 结构
- `web/style.css` — 样式
- `web/app.js` — 交互逻辑

### CSS 变量

定义 CSS 自定义属性统一管理颜色和间距：

```css
:root {
  --color-primary: #2563eb;
  --color-primary-dark: #1d4ed8;
  --color-primary-light: #3b82f6;
  --color-success: #16a34a;
  --color-danger: #ef4444;
  --color-text: #1e293b;
  --color-text-secondary: #64748b;
  --color-text-muted: #94a3b8;
  --color-bg: #f8fafc;
  --color-border: #e2e8f0;
  --color-card: #ffffff;
  --radius-sm: 8px;
  --radius-md: 10px;
  --radius-lg: 12px;
  --shadow-sm: 0 1px 3px rgba(0,0,0,0.06);
  --shadow-md: 0 4px 14px rgba(37,99,235,0.25);
  --shadow-lg: 0 8px 24px rgba(37,99,235,0.4);
  --sidebar-width: 64px;
}
```

### 动画关键帧

```css
@keyframes pulse { /* 呼吸脉冲光环 */ }
@keyframes shimmer { /* 光泽扫过 */ }
@keyframes spin { /* 旋转加载 */ }
@keyframes progress { /* 底部进度条 */ }
@keyframes blink { /* 指示灯闪烁 */ }
```

### JavaScript 改动

- 保持现有 API 调用逻辑不变
- 更新 DOM 操作以匹配新结构
- 添加侧边导航切换逻辑
- 按钮状态管理（加载态、自动模式态）

## 不变的部分

- 后端 API 接口不变
- 串口通信逻辑不变
- 视频流获取方式不变
- 历史记录 API 不变
- 配置管理 API 不变
